## 💡 프로젝트 소개
JB금융그룹 메인 페이지를 모작하기 위해 제작한 사이트입니다.

## ✅ 개발 기간
* 25.06.12 ~ 25.07.11

## ⚙️ 개발 환경
- 언어 : JSX / SCSS / JS
- 프레임워크 : Next.js
- 라이브러리 : React
- 도구 : VS Code / Git
- 배포 : Github Pages

## ✨ 주요 기능
- 반응형 웹 디자인
- Interactive UI/UX
  * 스크롤 이벤트
  * Intersection Observer 애니메이션
  * 마우스 오버 & 클릭 애니메이션
  * 관성 스크롤링
  * Draggable
